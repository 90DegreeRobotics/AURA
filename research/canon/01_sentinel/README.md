# Sentinel Security Docs (NRI Portable Bundle)

This directory is the **portable** Sentinel adoption bundle for product repositories.

## Canonical vs portable

| Role | Path |
| --- | --- |
| Canonical doctrine home | `C:\NRI\Sentinel\SENTINEL_IMPERVIOUS_PROTOCOL_MASTER_PLAN.md` |
| Portable plan copy | `docs/security/SENTINEL_IMPERVIOUS_PROTOCOL_MASTER_PLAN.md` (this folder) |

Product repos should copy this folder's plan plus the templates below (renamed without `.template`), or keep an explicit pointer to the canonical NRI path.
